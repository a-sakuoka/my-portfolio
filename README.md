# Portfolio Site

A minimal portfolio website built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lenis.

## Features

- Smooth scrolling with Lenis
- Beautiful animations with Framer Motion
- Minimal design with generous spacing
- Responsive layout
- Grain texture overlay for film-like quality

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash!!
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

- Update your name and title in `components/HeroSection.tsx`
- Modify the About section content in `components/AboutSection.tsx`
- Add your projects in `components/WorksSection.tsx`
- Update contact information in `components/Footer.tsx`
